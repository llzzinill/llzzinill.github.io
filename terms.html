Dreamshorts 서비스 이용 약관

본 약관은 Dreamshorts 앱(이하 '서비스') 이용과 관련하여 사용자와의 권리, 의무, 책임사항 등을 규정합니다.

1. 서비스 제공 목적

Dreamshorts는 사용자가 입력한 주제와 감정에 따라 자동으로 TikTok 영상을 생성하고 업로드하는 서비스입니다.

2. 사용자 책임

사용자는 Dreamshorts를 통해 생성된 콘텐츠가 타인의 권리를 침해하지 않도록 주의해야 하며, 모든 법적 책임은 사용자에게 있습니다.

3. 면책 조항

Dreamshorts는 AI 기반의 자동화된 영상 생성 도구로, 생성된 콘텐츠로 인해 발생하는 분쟁에 대해 책임을 지지 않습니다.

4. 약관 변경

본 약관은 필요 시 변경될 수 있으며, 변경 시 GitHub Pages를 통해 사전 공지됩니다.
