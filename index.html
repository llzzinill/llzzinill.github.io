Dreamshorts

AI 기반 TikTok 영상 자동 생성 플랫폼

Dreamshorts는 사용자가 입력한 주제와 감정 톤에 따라 TikTok 쇼츠 영상을 자동으로 생성해주는 서비스입니다.
GPT로 생성한 대본, TTS 음성, 배경 이미지와 자막, 썸네일까지 자동으로 조합해 완성된 영상을 TikTok API로 업로드할 수 있습니다.

📂 공식 문서

  • 서비스 이용 약관
  • 개인정보 처리방침
  • 도메인 인증 파일

🛠 주요 기능

  • GPT 기반 자동 대본 생성
  • Google TTS 음성 변환
  • PEXELS/Unsplash 기반 이미지 검색
  • MoviePy 기반 영상 합성
  • TikTok API 연동 자동 업로드
