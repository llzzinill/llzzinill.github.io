Dreamshorts 개인정보 처리방침

Dreamshorts는 사용자 개인정보 보호를 중요하게 생각하며, 다음과 같이 처리방침을 명시합니다.

1. 수집하는 정보

사용자 입력 주제, 선택한 감정, 생성된 콘텐츠의 메타 정보 등 최소한의 정보를 수집합니다.

2. 수집 목적

AI 기반 영상 생성 기능 제공, 사용자 맞춤 콘텐츠 개선을 위한 통계 분석 등

3. 정보 보관 및 삭제

수집된 정보는 생성 목적 외 사용되지 않으며, 요청 시 즉시 삭제됩니다.

4. 제3자 제공

Dreamshorts는 어떠한 경우에도 개인정보를 외부에 제공하지 않습니다.

5. 정책 변경

정책 변경 시 GitHub Pages를 통해 사전 공지됩니다.
